SISTEMA DE TICKETS SUPER MTV

Como usar:
1. Abra o arquivo index.html no navegador.
2. Cliente abre chamado pela tela principal.
3. O sistema gera um código TCK-000000.
4. O cliente consulta o status pelo código.
5. O suporte entra em "Painel suporte".
6. Senha padrão: 1234.
7. O suporte pode marcar: Em análise, Pedido realizado, Reparo feito, Não encontrado, Resolvido.

Observação:
Esta primeira versão salva tudo no navegador usando localStorage.
Para colocar online com vários atendentes/clientes reais, o próximo passo é ligar com banco de dados Supabase/Firebase.
